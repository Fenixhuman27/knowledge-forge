# Conectar USB Bluetooth

### Usuario

Tengo una computadora de escritorio AMD A4-3300 APU DE 64Bits que no tiene Bluetooth, quiero conectar un USB pero no me aparece en la PC. Busque varias soluciones pero aún no puedo hacer que aparezca el Bluetooth

### Asistente

Para conectar un dispositivo Bluetooth a tu computadora que no tiene Bluetooth incorporado, necesitas un adaptador USB Bluetooth. Si ya tienes uno, pero tu PC no lo reconoce, aquí hay algunos pasos que puedes seguir para solucionar el problema:

1. **Revisa la Conexión Física:**
   - Asegúrate de que el adaptador USB Bluetooth esté correctamente conectado a un puerto USB funcional de tu computadora.

2. **Verifica en el Administrador de Dispositivos:**
   - Abre el Administrador de Dispositivos (puedes buscarlo en el menú de inicio).
   - Busca en la sección "Bluetooth" o "Otros dispositivos" para ver si el adaptador aparece. Si tiene un icono de advertencia (triángulo amarillo), podría indicar un problema con los controladores.

3. **Instala los Controladores Correctos:**
   - La mayoría de los adaptadores Bluetooth vienen con un CD de instalación o un enlace para descargar los controladores necesarios. Asegúrate de instalar los controladores que correspondan a tu modelo de adaptador.
   - Si no tienes el CD, visita la página web del fabricante del adaptador y descarga los controladores más recientes para tu modelo y sistema operativo.

4. **Actualizar Controladores:**
   - En el Administrador de Dispositivos, haz clic derecho en el adaptador Bluetooth y selecciona "Actualizar controlador".
   - Elige "Buscar automáticamente software de controlador actualizado" para que Windows intente encontrar e instalar el controlador más reciente.

5. **Habilitar Bluetooth en Configuración:**
   - Una vez que el adaptador esté instalado correctamente, ve a "Configuración" > "Dispositivos" > "Bluetooth y otros dispositivos".
   - Asegúrate de que la opción Bluetooth esté activada.

6. **Reinicia tu Computadora:**
   - A veces, simplemente reiniciar tu computadora puede ayudar a que reconozca nuevos dispositivos.

7. **Prueba en Otro Puerto USB:**
   - Si el adaptador no aparece, intenta conectarlo a otro puerto USB para descartar un problema con el puerto original.

8. **Verifica Actualizaciones de Windows:**
   - Asegúrate de que tu sistema operativo esté completamente actualizado, ya que las actualizaciones pueden incluir mejoras y correcciones para la compatibilidad de hardware.

Si después de seguir estos pasos tu adaptador USB Bluetooth todavía no es reconocido, podría haber un problema con el adaptador en sí. Intenta probarlo en otra computadora para asegurarte de que funcione correctamente.
