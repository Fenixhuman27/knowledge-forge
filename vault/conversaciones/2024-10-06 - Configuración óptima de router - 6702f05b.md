# Configuración óptima de router

### Usuario

Puedes ayudarme con la mejor configuración para este router?

### Asistente

Para mejorar la configuración de tu router de Antel, aquí te dejo algunas sugerencias generales que puedes considerar:

1. **Cambio de SSID y Contraseña Wi-Fi**:
   - Cambia el nombre de la red Wi-Fi (SSID) por uno personalizado para evitar que otros identifiquen el tipo de router que usas.
   - Asegúrate de que la contraseña sea segura, usando una combinación de letras, números y símbolos.

2. **Actualización del Firmware**:
   - Verifica si hay actualizaciones de firmware disponibles en la opción de “Administración & Diagnóstico”. Un firmware actualizado mejora el rendimiento y la seguridad del router.

3. **Canales Wi-Fi**:
   - Si notas interferencias o lentitud en el Wi-Fi, ajusta el canal manualmente. Usa una aplicación para escanear redes Wi-Fi cercanas y seleccionar un canal menos saturado.

4. **Seguridad WPA3**:
   - Si tu router lo permite, configura la seguridad de la red Wi-Fi en WPA3 para una mayor protección. Si no está disponible, utiliza WPA2 como alternativa.

5. **Direcciones IP Estáticas**:
   - Si tienes dispositivos que necesitan conexión estable (como una consola de juegos o una Smart TV), asigna direcciones IP estáticas desde la configuración de DHCP.

6. **Control Parental**:
   - Si lo necesitas, configura filtros de control parental para gestionar el acceso de los dispositivos conectados.

7. **Gestión de Ancho de Banda (QoS)**:
   - Activa la opción QoS (Quality of Service) para priorizar ciertos dispositivos o aplicaciones que requieren más ancho de banda, como videollamadas o videojuegos en línea.

8. **Desactivar WPS**:
   - El WPS (Wi-Fi Protected Setup) puede ser un riesgo de seguridad. Es recomendable desactivarlo si no lo utilizas.

9. **Dispositivos Conectados**:
   - Desde la pantalla que me muestras, puedes ver una lista de los dispositivos conectados. Si detectas algún dispositivo no reconocido, cambia la contraseña y asegúrate de eliminar conexiones no deseadas.

10. **Red de Invitados**:
    - Si esperas visitas o necesitas que otros se conecten a tu Wi-Fi, habilita una red de invitados con acceso limitado, así proteges la red principal.
