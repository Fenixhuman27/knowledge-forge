## Qué es el CSS

El CSS (Cascading Style Sheets) es lo que define **cómo se ve tu web**.

👉 HTML = estructura (el “esqueleto”)  
👉 CSS = diseño (colores, tamaños, tipografías, espacios, animaciones)

Ejemplo simple:

- HTML dice: “esto es un título”
- CSS dice: “este título es grande, blanco, con sombra, centrado”

    const audio = document.getElementById("bgAudio");

    if (!audioPlaying) {
        audio.volume = 0.4;
        audio.play();
        audioPlaying = true;
    } else {
        audio.pause();
        audioPlaying = false;
    }
}
