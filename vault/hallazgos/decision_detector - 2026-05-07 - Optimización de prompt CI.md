# Decision Detector

Conversacion origen: [[2026-05-07 - Optimización de prompt CI - 69fc7219]]

## Hallazgos

- Solo tenemos que omitir el tema de crear un personaje, ya tenemos la banda y el disco está en todos lados... Vamos a usar este informe para crear una estrategia creo presupuesto para llevar nuestro propio proyecto a otro nivel lo más rápido posible! Lista?
- Ok, tenemos un problema y una posible solución! 
Yo no soy experto en todo lo que me acabas de explicar, así que tendrás que tener el control absoluto del proyecto para llevar adelante todo lo que me acabas de explicar en un formato de:
1)Vamos a hacer de a una cosa a la vez.
Luego de terminado eso, vamos al siguiente paso. 
2) Primero me das una hoja de ruta punto por punto y después vamos avanzando por puntos.
3) Decision ("Mario vamos a hacer esto, te explico")
Decisiones en conjunto (te doy mi punto de vista antes que crees toda la estrategia sobre ese punto)
4) Luego me das la estrategia desglosada para qur yo la ejecute tal cual con indicaciones simples de entender y seguir.
5) analizas antes de darme las instrucciones todas las herramientas que podamos usar para cada acción concreta (siempre gratis)
6) el objetivo es tener mínimo un avance diario.
